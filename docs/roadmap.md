# 路线图（Roadmap）

本文档描述项目的未来规划，**不代表当前版本已支持**。

---

## v0.2.0（下一版本）

计划内容：

- **中文文献手动导入** — 支持 BibTeX / RIS / CSV 格式导入本地中文文献，改善中文文献来源
- **报告生成模板配置化** — 允许用户自定义报告章节结构、图表样式、语言风格模板
- **更细粒度的 AI 变量选择预览** — 在发送给 AI 前展示每个变量的具体聚合摘要，而非仅显示风险等级
- **更好的图表管理** — 图表导出为 SVG/PNG、图表配色方案选择、图表尺寸控制
- **描述统计导出增强** — 支持将描述统计表导出为 Excel 格式
- **变量说明表编辑器** — 在 Web UI 中直接编辑变量说明（类型、标签、用途），无需重新上传 CSV

## v0.3.0

计划内容：

- **项目配置持久化** — 将分析配置（目标变量、分组变量、报告参数等）保存为项目文件（JSON/YAML），支持加载已有项目
- **多报告历史管理** — 保存生成报告的版本历史，支持对比和回看
- **更完善的错误监控** — 增加操作日志、错误追踪、使用统计
- **可选的本地模型支持** — 通过 Ollama / llama.cpp 等本地推理框架支持离线 AI 报告生成
- **自定义统计检验** — 允许用户选择更多统计方法（非参数检验、多重比较校正、效应量等）
- **国际化支持** — 界面英文版本，支持英文问卷分析

## v1.0.0（正式版）

计划内容：

- **用户权限与认证** — 基于密码或 OAuth 的登录系统
- **多用户隔离** — 每个用户独立的数据空间、分析历史和 API Key
- **生产部署方案** — 提供 Docker Compose 一键部署、Nginx 反向代理配置、HTTPS 设置指南
- **更完整的数据审计与合规说明** — 数据处理全链路可审计日志、数据保留策略、GDPR/个人信息保护法合规参考
- **数据持久化** — 可选数据库后端（SQLite / PostgreSQL），替代纯 session_state 方案
- **高级分析功能** — 加权调查数据处理、复杂抽样设计支持、纵向/面板数据分析

---

- 功能优先级可能根据社区反馈和实际使用场景变化。
