# 通用数据分析功能测试数据包

这个压缩包用于测试“通用问卷 / 表格数据 AI 辅助统计分析与报告生成平台”。

## 文件说明

1. `01_课程教学评价数据_含变量说明.xlsx`
   - 中文字段名
   - 有变量说明表
   - 适合测试：中文变量识别、量表题、有序分类、核心满意度分析、报告生成

2. `02_博物馆观众体验调查_无变量说明.xlsx`
   - 没有变量说明表
   - 工作表名不是固定名称
   - 适合测试：自动识别工作表、自动推断变量类型、日期变量、无变量表时的分析流程

3. `03_员工满意度调查_含缺失和英文变量.xlsx`
   - 英文字段名
   - 有变量说明表
   - 含缺失值、文本变量、英文分类变量
   - 适合测试：变量说明表覆盖、缺失值统计、英文变量识别、回归分析

4. `04_电商用户转化行为_无变量说明.csv`
   - CSV 文件
   - 二分类结果变量 converted
   - 高基数 ID 变量 user_id
   - 适合测试：CSV 上传、二分类变量、数值变量、分类变量、转化分析

## 建议测试顺序

先用 01 测“正常问卷数据”，再用 02 测“没有变量说明表也能跑”，然后用 03 测“英文变量+缺失值”，最后用 04 测“CSV 和二分类结果变量”。

## 注意

这些都是模拟数据，只用于功能测试，不代表真实调查结论。
